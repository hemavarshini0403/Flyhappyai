   - Logic based on complaint type, keywords, duration, and sentiment

   - Allow complaint input in English and optionally translate/support other Indian languages

   - Help users file complaints
   - Suggest complaint category and needed documents
   - FAQ support for airline grievance process

7. Complaint tracking dashboard
   - Show complaint ID, status, progress bar, timestamps
   - Status flow: Submitted → Under Review → In Process → Resolved / Rejected

8. Admin panel
   - View all complaints
   - Filter by airline, category, priority, date, status
   - Update complaint status
   - View uploaded files and extracted OCR text
   - Analytics dashboard with charts

9. Reports and analytics
   - Complaint count by type
